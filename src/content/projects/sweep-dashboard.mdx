## What it does

A visualization dashboard for exploring hyperparameter sweep results across ML experiments. Designed to help researchers quickly identify the best-performing configurations and understand how different hyperparameters interact.

## Key features

- Interactive plots for comparing sweep runs side-by-side
- Filter and sort by any metric or hyperparameter
- Visual identification of trends and parameter interactions
- Fast iteration over large sweep result sets
