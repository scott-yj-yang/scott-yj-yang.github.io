## What it does

Estrapatch is a scheduling and reminder tool for managing estrogen patch changes. It tracks patch rotation schedules and sends timely reminders to help maintain consistent dosing.

## Key features

- Customizable rotation schedules
- Reminder notifications
- History tracking and logging
