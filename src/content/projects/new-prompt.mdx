## What it does

A tool for configuring and customizing terminal prompts. Quickly set up an informative, clean shell prompt with git status, environment info, and custom styling.

## Key features

- Git branch and status integration in prompt
- Environment and virtualenv awareness
- Fast — no noticeable delay on prompt render
- Easy to configure and extend
