## Overview

CanvasGroupy automates the creation and management of student groups on Canvas LMS. Built during undergraduate research at the Fleischer Research Lab, it streamlines the repetitive work that instructors and TAs face when organizing large courses.

## Key features

- Automated group creation and enrollment via the Canvas API
- Configurable group assignment strategies
- Batch operations for large courses
- Integration with Canvas gradebook
